# Analytics

## What Analytics Tracks

Analytics records successful storefront size-guide opens.

This helps merchants understand:
- which size tables are opened most
- which products trigger the most guide opens
- when shoppers interact with size guides

## What Analytics Shows

- total guide opens
- unique products viewed
- unique tables opened
- date-range trend
- top size tables
- top products
- period comparison
- best day
- best hour
- best weekday + hour
- hourly breakdown
- recent events

## Filters

Analytics supports:
- date range filtering
- size-table drilldown
- recent-event date/time filtering

## CSV Export

Merchants can export chart and product rankings as CSV for external reporting.

## Recent Events

Recent events show:
- product name
- product handle
- size table opened
- timestamp

Recent events support:
- row numbering
- pagination
- event-only date/time filtering

## Best Use Cases

Use Analytics to:
- identify products where fit guidance matters most
- find heavily used size tables
- compare performance across periods
- spot peak shopping times for size-guide usage

## What To Be Careful With

- Analytics data starts only after tracking is active and the migration is applied.
- Low event volume may produce weak trend patterns.
- Hour/day insights reflect recorded server timestamps.
