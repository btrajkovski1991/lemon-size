# Getting Started

## What Lemon Size Is

Lemon Size is a Shopify embedded app for creating size tables, assigning them to products, and showing them through a storefront product-page modal.

It helps merchants:
- create reusable size tables
- match the right table to the right product
- show guide text and images on storefront
- let shoppers switch supported units
- review analytics for size-guide usage

## Main App Areas

- `Home`
  - onboarding checklist
  - preview matched chart
  - rule conflict explanation
- `Assignments`
  - direct product and catalog matching
- `Keyword rules`
  - fallback matching rules
- `Rule tester`
  - manual matching simulation
- `Size tables`
  - chart builder and templates
- `Analytics`
  - size-guide usage insights
- `Help`
  - setup and support notes

## Basic Setup Flow

1. Install Lemon Size on your Shopify store.
2. Create one or more size tables in `Size tables`.
3. Add direct assignment rules in `Assignments`.
4. Add keyword fallback rules only if needed.
5. Open `Online Store > Themes > Customize`.
6. Add the Lemon Size app block to the product template.
7. Open a product page and confirm the size guide loads correctly.

## Important Setup Note

Lemon Size currently uses an app block on the product template.

It does not appear in Shopify under `App embeds`.

## What To Check On Your First Storefront Test

- the size-guide button appears
- the correct table opens
- guide image appears if enabled
- units switch correctly
- only relevant size rows are shown when variant filtering is available
