# FAQ

## Does Lemon Size use App Embeds?

No. Lemon Size currently uses an app block on the product template.

## Can shoppers switch units on storefront?

Yes. Supported unit switching includes:
- `MM`
- `CM`
- `INCHES`

## Can Lemon Size filter rows based on product variants?

Yes. Lemon Size can try to show only relevant rows based on the product’s available variant values.

## Do I need keyword rules?

Not always.

Best practice:
- use direct assignments first
- use keyword rules only as fallback

## Can I reuse one size table on many products?

Yes. One table can be assigned to many matching rules.

## Can I duplicate an existing table?

Yes. Duplicated tables open directly in the editor and can be saved as a new table.

## Can I import a table instead of typing every row?

Yes. Lemon Size supports CSV and pasted table import.

## Can guide images be hidden without deleting them?

Yes. A guide image can remain saved while being hidden on storefront.

## Does Lemon Size support jewelry sizing?

Yes. Lemon Size supports:
- rings
- necklaces
- bracelets

It also supports `mm` as a base unit and converts jewelry measurement columns like:
- `DIAMETER`
- `CIRCUMFERENCE`
- `FINGER SIZE`
- `NECKLACE LENGTH`
- `BRACELET LENGTH`
