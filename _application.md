# Lemon Size Application Summary

## App name

Lemon Size

## What it does

Lemon Size helps Shopify merchants create custom size charts, assign them to products, and display them directly on product pages. It supports rule-based chart matching, storefront unit switching, guide images, and analytics for size-guide usage.

## Key features

- Create custom size tables with manual editing, CSV or paste import, and built-in starter templates
- Assign size tables to products, collections, vendors, product types, tags, and fallback keyword rules
- Display size guides on storefront product pages with guide images and shopper-friendly unit switching
- Automatically filter visible size rows to match the product's real available variants where possible
- Support `cm`, `mm`, and `in` conversions for measurement-based size tables
- Built-in templates for categories such as rings, necklaces, and bracelets
- Analytics page showing top viewed tables, top products, trends, best day/hour, and recent guide opens

## How it works

1. Install the app in a Shopify store
2. Create one or more size tables inside Lemon Size
3. Assign a size table to products or create fallback keyword rules
4. Add the Lemon Size app block to the product template in the Shopify theme editor
5. The matching size guide displays automatically on product pages

## Who it is for

- Fashion brands
- Shopify stores selling apparel, footwear, jewelry, accessories, or pet products
- Dropshipping stores
- Merchants who want better size guidance without editing theme code manually

## Benefits

- Reduce returns caused by size uncertainty
- Improve conversion rate by helping shoppers choose the right size
- Improve the storefront user experience with clear size guidance
- Save merchant setup time with reusable rules, templates, and imports
- Provide marketing insight through analytics on size-guide engagement

## Current functionality included

- Size table management
- Product and collection assignments
- Keyword fallback rules
- Storefront chart rendering through a theme app block
- Guide image support
- Unit conversion for supported measurement columns
- Variant-aware row filtering
- Rule preview and rule tester tools
- Analytics tracking and reporting
- Merchant help and support documentation

## How merchants manage the app

- `Home`: setup checklist, product preview, and matching overview
- `Assignments`: connect charts to products, collections, vendors, types, and tags
- `Keyword rules`: define fallback matching when no direct assignment exists
- `Rule tester`: manually test matching logic before checking the storefront
- `Size Charts`: create, edit, duplicate, import, and manage size tables
- `Analytics`: review table opens, product engagement, trends, and recent events
- `Help`: setup guidance, troubleshooting, and support details

## Suggested long app summary

Lemon Size is a Shopify size-guide app built for merchants who want clear, flexible, and easy-to-manage sizing information on their storefront. Merchants can create custom size tables, assign them directly to products or collections, define fallback keyword rules, display guide images, and let shoppers switch between supported measurement units.
